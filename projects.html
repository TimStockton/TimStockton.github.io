Projects (Software Engineering & Web Development)

My largest personal software engineering project by far is the Psidemica website. This site is an artist portfolio for my own music and animations, but it is incredibly feature rich. It contains an entire e-commerce platform as well as a members activity hub with accounts, messaging, games, a quote generator, and an interface for a NASA API. I also built admin tools and functionality. I used CSS, HTML, and JavaScript as is expected of a website, and I used PHP for the structure and logic. I also built automated testing tools using Python. I coded all of it from scratch, starting with a textbook example and modifying it beyond recognition, and did not use any frameworks. Music is what keeps me going, life would be so dull without it. I have been producing since 2013, and love being able to artistically express myself.

Calculator

I created a basic calculator using JavaScript.

It doesn't handle engineering notation or any advanced features like trigonometry, logarithms, and exponentiation. I might add these one day though.

I plan on restyling it so it isn't so grey and sad.

I created a simple temperature converter that outputs values in Fahrenheit, Celcius, and Kelvin based on the value you enter.

The outputs are tied together, and change as soon as one of the values change. Conversion is automatic.


I created a ROT13 Encoder/Decoder, also known as a Shift Cipher or a Caesars Cipher. It takes the input, and shifts each letter provided by thirteen places in the alphabet. It can be used to send and recieve basic secret messages.


I created a Roman numeral converter too. You can enter a Roman numeral, and it will output the Arabic number equivalent. You can also enter an Arabic number, and it will output the Roman numeral equivalent. Comes in handy for the Superbowl.

I have also worked on a drum machine, markdown previewer, and survey form. Currently, I need to move the code and restyle them so they can fit into this portfolio.
