Test plan for user_management.sh

ensures all commands, options, and variations function correctly
all tests must be run with effective root privileges
steps and tests should be done in order

-------------------------
Setup
-------------------------

Install script at:
/usr/local/sbin/user_management.sh

Ensure root is the owner and group for security:
chown root:root /usr/local/sbin/user_management.sh

Ensure the script is executable by root only:
chmod 700 /usr/local/sbin/user_management.sh

Create a CSV file (used to create test users):
/tmp/test_users.csv
"
username,home,full_name,password,expiration_date
testuser1,,Test User One,password123,2026-01-01
testuser2,/customhome/jeff,Jeff Johnson,pass456,
nopassuser,,No Password,,2027-01-01
expireuser,,Patricia Past,Str0ngerPa$$w0rd,2000-02-02
root,,Admin Istrator,root123,
"
root is included to verify script refusal behavior

-------------------------
Logging Tests
-------------------------

*** Log File Creation

Command:
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log report_no_pass

Expected:
log file is created (if missing)
console output appears
identical output appended to log file

*** Log File Overwrite/Append

Command (running the above command a second time):
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log report_no_pass

Expected:
second run appends to log file, does not overwrite

-------------------------
User Creation Tests
-------------------------

*** Create Users from File

Command:
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log add_users_from_file /tmp/test_users.csv

Expected:
users created with specified home or default
passwords set where provided
forced password change on login
expiration set where provided
log output indicates successful user creations
log output indicates root was skipped

*** Interactive User Creation

Command:
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log add_user testuser3

provide prompted values

Expected:
user created with correct home dir
password set
forced password change on login
expiration date set if provided
log output indicates successful user creation

Expected:
script identifies nopassuser

*** report_expirations

Command:
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log report_expirations

-------------------------
Account Modification Tests
-------------------------

*** Lock Users

Command:
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log lock testuser1 testuser2

Expected:
accounts locked (! prefix in /etc/shadow)
log entries present

*** Unlock Users

Command:
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log unlock testuser1 testuser2

Expected:
accounts unlocked
log entries present

*** Change Expiration (valid user)

Command:
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log set_expire testuser3 2030-01-01

Expected:
expiration updated
log entry recorded

*** Change Expiration (root)

Command:
sudo /usr/local/sbin/user_management.sh log /var/log/user_management_test.log set_expire root 2030-01-01

Expected:
script refuses
no changes made
