Assignment 2: Clean List Shell Script

Project Overview

Write a shell script to concatenate lists together and output the resulting list. The script should clean the list of any redundant items while preserving the original order.

Requirements

  • Remove Redundancy: Do not include any argument that is a sub-list of the entire list. The script will clean the list of any redundant items.
  • Preserve Order: Maintain the original order of the list items.
  • Handle Special Cases: Process special colon patterns as described below.
  • No Temporary Files: Do not use temporary files in your implementation.
Important: Projects using temporary files will not be graded.

Special Cases

Your script must handle these special colon patterns:

  • A : at the beginning of the list is the same as a . at the beginning
    Example: :/bin is the same as .:/bin
  • A :: anywhere in the list is the same as :.:
    Example: /bin::/etc is the same as /bin:.:/etc
  • A : at the end of the list is the same as a :.
    Example: /bin: is the same as /bin:.
Grading Note: If you do not handle these special cases, the best grade you can receive for the assignment is an A-.

Input and Output

Input: Colon or space separated lists

Output: A single colon separated list with original order preserved and all redundant items removed

Usage

clean_list list ...

Where list is a colon or whitespace separated list.

Examples

Example 1: Basic List Concatenation

Command:
prompt> clean_list a a:b a:b:c :x: y:z

Output:
a:b:c:.:x:y:z

Example 2: Path-like Lists

Command:
prompt> clean_list /bin:/usr/bin:/usr/openwin/bin /usr/bin:/usr/etc:/etc: /usr/bin/X11 .:/bin

Output:
/bin:/usr/bin:/usr/openwin/bin:/usr/etc:/etc:.:/usr/bin/X11

Example 3: Mixed Separators

Command:
prompt> clean_list apple:orange:apple pear orange peach

Output:
apple:orange:pear:peach

Implementation Notes

  • The script should be named clean_list
  • Make sure to handle both colon-separated and space-separated input lists
  • Process special colon cases before removing duplicates
  • Maintain the order of first occurrence when removing duplicates
  • Test thoroughly with the provided examples
