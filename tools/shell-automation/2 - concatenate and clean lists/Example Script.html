Example Script for Project 2

The following script is very similar to Project 2; however, the major differences are:

  • It works with a single variable, not the command line arguments
  • The script tests to see if directories exist (not needed for Project 2)
  • For Project 2, each positional parameter can be a special case:
    • A : at the beginning of a parameter is really .:
    • A :: in a parameter is really :.:
    • A : at the end of a parameter is :.

You should run this script and see what happens, then modify it to meet the requirements for Project 2.

The tricky part is that A:B :C should become A:B:.:C, but the code below won't handle that case.

#!/bin/sh

NP=
for P in `echo $PATH | sed -e 's/^:/.:/' -e 's/::/:.:/' -e 's/:$/:./' -e 's/:/ /g'`
    do
    case $NP in  
    "")
        if [ -d "$P" ]
	    then
            NP="$P"
            fi 
    ;;
    $P|$P:*|*:$P:*|*:$P)
        continue
        ;;
    *)  
        if [ -d "$P" ]
	    then
            NP="$NP:$P"
            fi
        ;;
        esac
    done
echo $NP
