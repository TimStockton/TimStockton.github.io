Grading Rubric: clean_list Shell Script Assignment

Course: Shell Programming

Assignment: clean_list Script

Total Points: 100

Assignment Weight: Major Project

Assignment Overview

Students must write a shell script that processes colon and space-separated lists, removes duplicates while preserving order, and handles special cases involving colons and dots. The script must concatenate input lists, clean redundant items, and output a colon-separated list with original order preserved.

Learning Objectives

Upon successful completion of this assignment, students will demonstrate ability to:

  • Process command-line arguments in shell scripts
  • Manipulate string data and list structures
  • Implement deduplication algorithms while preserving order
  • Handle special case conversions as specified
  • Write robust shell scripts without using temporary files
