About

I enjoy web development, machine learning, software engineering, data analysis, and all things music. I'm passionate about the internet and understanding complex systems, and I love to create. When I'm not programming, I like to produce electronic bass music, DJ, make 3D animations, play video games, hike, lift weights, and write blog posts.

My programming journey began in 2019 learning the basics with Free Code Camp where I completed three certifications: Responsive Web Design, JavaScript Algorithms & Data Structures, and Front End Development Libraries. Some of the projects on this site were made as part of the certificate curriculum.

Now I have a bachelors degree in computer science from Keene State College (graduated in 2023 with double honors, Computer Science major and Mathematics minor). The first full-stack application I worked on was a college research project called Owl Assist Information System (OAIS). I was one of the first 3 student devs, and I assisted in it's construction and was tasked with maintaining it. It is a Flask application that utilizes web scrapers, Dialog Flow, Google Cloud, AWS, Selenium, and more. This project was my first real exposure to Git and Github as well as Trello. There were a couple articles written about OAIS. The first was in The Equinox: April 14, 2021, and the second was in KSC Stories: April 16, 2021. There was also an Equinox Video Interview: April 29, 2021.

Currently, I am working on personal projects and challenges to learn more and keep my skills sharp.
