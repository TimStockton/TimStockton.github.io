About Me

I enjoy web development, machine learning, software engineering, data analysis, and all things music. I'm passionate about the internet and understanding complex systems, and I love to create. I'm also a veteran, having honorably served five years in the Marine Corps, so I like to stay active. When I'm not programming or producing electronic bass music, I like to DJ, make 3D animations, play video games, run, hike, lift weights, and write blog posts.

My programming journey began in 2019 learning the basics with Free Code Camp where I completed three certifications: Responsive Web Design, JavaScript Algorithms & Data Structures, and Front End Development Libraries. Some of the projects on this site were made as part of the certificate curriculum.

Now I have a bachelors degree from Keene State College (graduated in 2023 with double honors, Computer Science major and Mathematics minor). The first full-stack application I worked on was a college research project called Owl Assist Information System (OAIS). I was one of the first 3 student devs, and I assisted in it's construction and was tasked with maintaining it. It is a Flask application that utilizes web scrapers, Dialog Flow, Google Cloud, AWS, Selenium, and more. This project was my first real exposure to Git and Github as well as Trello. There were a couple articles written about OAIS. The first was in The Equinox: April 14, 2021, and the second was in KSC Stories: April 16, 2021. There was also an Equinox Video Interview: April 29, 2021.

I have since worked professionally as a full-stack software engineer (you can view my job experience here), interacting with incredibly large code repositories and databases. I am particularly drawn to back-end software engineering, and I am currently taking courses to prepare for the AWS Solutions Architect and CompTIA Linux+ certification exams. I am also working on improving this portfolio.
