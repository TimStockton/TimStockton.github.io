Calculator

I created a basic calculator using JavaScript.

It doesn't handle engineering notation or any advanced features like trigonometry, logarithms, and exponentiation. I might add these one day though.

I plan on restyling it so it isn't so grey and sad.

I created a simple temperature converter that outputs values in Fahrenheit, Celcius, and Kelvin based on the value you enter.

The outputs are tied together, and change as soon as one of the values change. Conversion is automatic.


I created a ROT13 Encoder/Decoder, also known as a Shift Cipher or a Caesars Cipher. It takes the input, and shifts each letter provided by thirteen places in the alphabet. It can be used to send and recieve basic secret messages.


I created a Roman numeral converter too. You can enter a Roman numeral, and it will output the Arabic number equivalent. You can also enter an Arabic number, and it will output the Roman numeral equivalent. Comes in handy for the Superbowl.
